# 🧠 SMART Resume Analyzer

AI-powered resume analysis tool with smart candidate screening and automated email notifications.

## Features
- 📄 Resume parsing (PDF, DOCX, TXT)
- 🤖 AI-generated content detection
- 📊 Skill matching & scoring
- 📧 Automated email notifications
- 🎯 Smart candidate ranking

## Tech Stack
- Flask + SQLAlchemy
- PyMuPDF + python-docx
- NLTK + scikit-learn
- TextBlob for NLP

## Deployment
This project is configured for deployment on Hugging Face Spaces with Gradio SDK.
